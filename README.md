Các chức năng chính trong project:
1. Thành phần lương (TPL)
- Hiển thị danh sách TPL, lọc danh sách, phân trang, tùy chỉnh cột
- Sửa trạng thái và xóa nhiều TPL cùng lúc
- Thêm, nhân bản, sửa, xóa bản ghi TPL
- Thêm nhanh TPL từ TPL hệ thống
2. Thành phần lương hệ thống (TPLHT)
- Hiển thị danh sách TPLHT, lọc danh sách, phân trang, tùy chỉnh cột
- Đưa nhiều TPLHT vào sử dụng cùng lúc
